Detailed Description

Sketch for the RS485 Bus Master which controlls a chain of RS485 Slaves.

Author
Balz Reber
Date
17.12.2019

This is the Sketch for the RS485 Bus Master. The RS485 bus Master has to be an Arduino Mega. This Arduino Mega is connected to the PC running DCS-bios via USB. There can't be anything connected to that Arduino except the RS485 Bus. Preferably this runs on an ABSIS MEGA with a Bus termination resistor installed on the RS485 Bus. You can find more information about the RS485 Bus System Architecture in the OH documentation.
