Detailed Description

This is the OpenHornet Sketch Template.

Author

Date

This is the Open Hornet Sketch Template. It should be used as a starting point for every new sketch. Please copy the whole OHSketchTemplate folder to start. As it also contains some test skip files needed for travis. After that, please replace all the @tags with your own content.
